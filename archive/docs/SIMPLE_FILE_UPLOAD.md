# Simple File Upload Guide

## ✅ Easy Way - No Setup Needed!

### How to Add Files to Products:

1. **Upload file to Google Drive**:
   - Go to: https://drive.google.com/drive/folders/1bTLjx1RVZRrOapo1rd81iepFrye6SXIY
   - Click "New" → "File upload"
   - Select your PDF or CDR file
   - Wait for upload to complete

2. **Get the shareable link**:
   - Right-click the uploaded file
   - Click "Get link"
   - Make sure it's set to "Anyone with the link can view"
   - Click "Copy link"

3. **Paste in Product Form**:
   - Go to your Product Form (New or Edit)
   - Find "Artwork PDF" or "Artwork CDR" field
   - Paste the Google Drive link
   - Save the product

4. **Done!**
   - The link saves in the database
   - The link syncs to all related orders automatically
   - Anyone can view the file by clicking the link

## Example:

**Good link format:**
```
https://drive.google.com/file/d/1ABC123xyz/view?usp=sharing
```

**Also works:**
```
https://drive.google.com/open?id=1ABC123xyz
```

## Benefits:

✅ No complex setup
✅ Files stored in your Google Drive
✅ Easy to manage and organize
✅ Links sync to orders automatically
✅ Can update files anytime in Drive

## Tips:

- Organize files by product name in the Drive folder
- Use clear file names like "ProductName_Artwork.pdf"
- Keep the Drive folder organized with subfolders if needed
- Old links still work even if you rename files in Drive
